## Quick Start

<Steps>
  <Step title="Create a Meta App">
    Go to [developers.facebook.com](https://developers.facebook.com) and create a new app
  </Step>
  <Step title="Get Access Token">
    Use the Graph API Explorer to generate an access token with required permissions
  </Step>
  <Step title="Make API Calls">
    Use the token to authenticate your API requests
  </Step>
</Steps>
