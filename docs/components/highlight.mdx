Highlight important text with different visual styles and colors.

## Marker Style (Default)

The default highlight style resembles a marker pen:

```jsx
<Highlight>highlighted text</Highlight>
```

The <Highlight>Magic UI Highlighter</Highlight> makes important <Highlight color="blue">text stand out</Highlight> effortlessly.

## Underline Style

A clean underline effect:

```jsx
<Highlight variant="underline" color="orange">underlined text</Highlight>
```

The <Highlight variant="underline" color="orange">Magic UI Highlighter</Highlight> makes important <Highlight variant="underline" color="blue">text stand out</Highlight> effortlessly.

## Box Style

Text with a subtle background box:

```jsx
<Highlight variant="box" color="purple">boxed text</Highlight>
```

This is <Highlight variant="box" color="purple">important boxed text</Highlight> in your content.

## Gradient Style

A gradient background effect:

```jsx
<Highlight variant="gradient" color="pink">gradient highlight</Highlight>
```

Check out this <Highlight variant="gradient" color="pink">gradient highlight</Highlight> effect.

## Shorthand Components

Use `Marker` or `Underline` as convenient shortcuts:

```jsx
<Marker>yellow marker</Marker>
<Marker color="blue">blue marker</Marker>
<Underline>orange underline</Underline>
<Underline color="pink">pink underline</Underline>
```

<Marker>yellow marker</Marker> and <Marker color="blue">blue marker</Marker> and <Underline>orange underline</Underline> and <Underline color="pink">pink underline</Underline>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | string | marker | Style: marker, underline, box, gradient |
| `color` | string | yellow | Color: yellow, blue, green, pink, orange, purple, red |
| `animated` | boolean | false | Enable pulse animation |
| `className` | string | - | Custom CSS classes |

## Shorthand Components

### Marker

Shorthand for `<Highlight variant="marker">`:

```jsx
<Marker color="blue">highlighted</Marker>
// Equivalent to:
<Highlight variant="marker" color="blue">highlighted</Highlight>
```

### Underline

Shorthand for `<Highlight variant="underline">` with orange as default color:

```jsx
<Underline color="pink">underlined</Underline>
// Equivalent to:
<Highlight variant="underline" color="pink">underlined</Highlight>
```
