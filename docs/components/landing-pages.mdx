DevDoc provides a set of components to build beautiful custom landing pages for your documentation. These components allow you to create hero sections, feature grids, CTAs, and moreâ€”all using MDX.

## Setting Up a Custom Page

To create a custom landing page, add these frontmatter options to your MDX file:

```yaml
---
title: My Landing Page
description: A custom landing page
mode: custom
hideHeader: true
---
```

| Property | Type | Description |
|----------|------|-------------|
| `mode` | `'default' \| 'wide' \| 'custom'` | Set to `custom` for full-width landing pages |
| `hideHeader` | `boolean` | Hides the default page header |
| `fullWidth` | `boolean` | Removes max-width constraints |
| `background` | `string` | Custom background CSS value |

## Components

### Hero

A full-width hero section for the top of your landing page.

```jsx
<Hero variant="dark" minHeight="lg" align="center">
  {/* Your content here */}
</Hero>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'gradient' \| 'dark' \| 'pattern'` | `'default'` | Background style |
| `align` | `'top' \| 'center' \| 'bottom'` | `'center'` | Vertical alignment |
| `minHeight` | `'sm' \| 'md' \| 'lg' \| 'full'` | `'md'` | Minimum height (40vh, 60vh, 80vh, 100vh) |

**Variants:**
- `default` â€” Standard background color
- `gradient` â€” Subtle gradient from top to bottom
- `dark` â€” Dark background with white text
- `pattern` â€” Radial gradient pattern

### Center

Centers content horizontally with optional vertical spacing between children.

```jsx
<Center gap="lg">
  <Tagline>WELCOME</Tagline>
  <Headline>Hello World</Headline>
  <Description>Some description text</Description>
</Center>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `gap` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Vertical spacing between children |

---

### Tagline

Uppercase text for subtitles or category labels.

```jsx

</Tagline>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'muted' \| 'accent'` | `'default'` | Text color variant |
| `tracking` | `'normal' \| 'wide' \| 'wider' \| 'widest'` | `'wider'` | Letter spacing |


## Using Custom HTML

You can also use custom HTML/JSX directly in your MDX for advanced layouts:

```jsx
<div className="w-full flex items-center justify-center">
  <pre 
    className="text-sm font-mono text-white"
    style={{
      fontFamily: "'Courier New', monospace",
      lineHeight: 1.1,
    }}
  >
{`Your ASCII art or custom content here`}
  </pre>
</div>
```

<Callout type="tip">
  Use Tailwind CSS classes directly in your MDX for quick styling adjustments.
</Callout>

## Tips

1. **Keep it simple** â€” Start with the basic components and add complexity as needed
2. **Test responsively** â€” All components are responsive, but test on different screen sizes
3. **Use semantic sections** â€” Use `Section` with `id` props for anchor navigation
4. **Consistent spacing** â€” Use `Spacer` components for consistent vertical rhythm
5. **Dark sections** â€” Use `variant="dark"` on `Hero` and `Section` for contrast
