Cards are great for navigation and highlighting key content.

## Basic Card

```jsx
<Card title="Getting Started" icon="rocket" href="/quickstart">
  Learn how to get up and running quickly.
</Card>
```

<Card title="Getting Started" icon="rocket" href="/quickstart">
  Learn how to get up and running quickly.
</Card>

## Card with Header Image

Use `img` for a full-width header image:

```jsx
<Card title="Dashboard" img="/images/dashboard.png" href="/dashboard">
  View your analytics and insights.
</Card>
```

## Custom Icon Image

Use a React node in `icon` for custom images:

```jsx
<Card 
  title="Claude Code" 
  icon={<img src="/assets/images/claude.png" className="h-8 w-8 object-contain" />}
>
  Custom skills with slash commands.
</Card>
```

Control the size with Tailwind classes:

```jsx
{/* Small */}
<Card icon={<img src="/logo.png" className="h-6 w-6" />} title="Small">...</Card>

{/* Large */}
<Card icon={<img src="/logo.png" className="h-12 w-12" />} title="Large">...</Card>

{/* Rounded */}
<Card icon={<img src="/logo.png" className="h-10 w-10 rounded-full" />} title="Rounded">...</Card>
```

## Horizontal Card

```jsx
<Card title="Documentation" icon="book" horizontal>
  Complete guides and API references for developers.
</Card>
```

<Card title="Documentation" icon="book" horizontal>
  Complete guides and API references for developers.
</Card>

## Card Group

Display cards in a responsive grid:

```jsx
<CardGroup cols={2}>
  <Card title="First Card" icon="star">
    Description for the first card.
  </Card>
  <Card title="Second Card" icon="gear">
    Description for the second card.
  </Card>
</CardGroup>
```

<CardGroup cols={2}>
  <Card title="First Card" icon="star">
    Description for the first card.
  </Card>
  <Card title="Second Card" icon="gear">
    Description for the second card.
  </Card>
</CardGroup>

<CardGroup cols={3}>

## Props

### Card Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | string | required | Card title |
| `icon` | string \| ReactNode | - | Phosphor icon name or custom React element (e.g., `<img />`) |
| `iconSize` | `sm` \| `md` \| `lg` \| `xl` | `md` | Icon container size (32px, 40px, 48px, 64px) |
| `href` | string | - | Link destination |
| `horizontal` | boolean | false | Horizontal layout |
| `img` | string | - | Full-width header image URL |
| `cta` | string | - | Call-to-action text |
| `arrow` | boolean | - | Show arrow icon (auto for external) |

### CardGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `cols` | number | 2 | Number of columns (1-4) |
