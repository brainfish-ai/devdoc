## Headings

Use `#` symbols for headings:

```md
# Heading 1
## Heading 2
### Heading 3
```

## Text Formatting

| Style | Syntax | Result |
|-------|--------|--------|
| Bold | `**bold**` | **bold** |
| Italic | `_italic_` | _italic_ |
| Code | `` `code` `` | `code` |
| Strike | `~~strike~~` | ~~strike~~ |

## Lists

**Unordered:**
- Item one
- Item two
- Item three

**Ordered:**
1. First
2. Second
3. Third

## Links

```md
[Link text](https://example.com)
[Internal link](/quickstart)
```

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Code Blocks

Use triple backticks with a language identifier:

```javascript
function hello() {
  console.log("Hello, world!");
}
```

## Tables

```md
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
```

## Components

Beyond standard Markdown, you can use custom components:

<Note>
This is a note callout.
</Note>

<Tip>
This is a tip callout.
</Tip>

<Warning>
This is a warning callout.
</Warning>
