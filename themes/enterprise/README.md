# Enterprise Theme

Professional theme designed for enterprise documentation needs.

## Features

- Professional blue color scheme
- IBM Plex font family
- Enhanced typography hierarchy
- Polished UI elements
- Wider content area

## Usage

```json
{
  "$schema": "https://devdoc.sh/theme.json",
  "extends": "github:brainfish-ai/devdoc/themes/enterprise"
}
```

## Best For

- Enterprise API documentation
- B2B SaaS products
- Professional services
- Corporate documentation

## Customization

Brand it with your company colors:

```json
{
  "$schema": "https://devdoc.sh/theme.json",
  "extends": "github:brainfish-ai/devdoc/themes/enterprise",
  "colors": {
    "primary": "#your-brand-color"
  }
}
```
